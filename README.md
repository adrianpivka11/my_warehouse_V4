# My Warehouse V4

Educational Python project focused on Object-Oriented Programming, SQL, and software development fundamentals.

This project is a simple warehouse management system that simulates basic functionality of inventory software.

It was created as a school project to practice connecting **Python**, **SQL**, and programming concepts used in real business applications.

---

## Overview

The application allows management of warehouse data such as products, stock records, and related operations.

The main goal of the project was learning how to design a structured application using object-oriented principles and database integration.

---

## Key Features

- Manage warehouse items and inventory records
- Add, edit, update, and remove data
- Work with SQL database
- Use Python modules and structured project files
- Apply Object-Oriented Programming principles
- Simulate real warehouse software logic
